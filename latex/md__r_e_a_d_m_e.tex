\begin{quote}
{\bfseries{Задача 33:\+}} ~\newline
 \end{quote}


\begin{quote}
Однажды математик С. Улам разделил лист бумаги на клетки и, написав в центре 1, начал писать по спирали против часовой стрелки все натуральные числа подряд, выделяя простые числа. Скоро простые числа выстроились в довольно-\/таки закономерном порядке, образуя интересный узор. Этот узор позже стал объектом исследования и получил название скатерть Улама. ~\newline
 {\bfseries{Составьте программу, демонстрирующую скатерть Улама размером 100 × 100 клеток (вместо простых чисел выводите звездочку “\texorpdfstring{$\ast$}{*}”).}} \end{quote}


Дополнительно в рамках задания реализовано клиент-\/серверное взаимодействие:\+ ~\newline

\begin{DoxyItemize}
\item Сервер принимает команды от клиента. ~\newline

\item По команде {\ttfamily -\/s} генерирует скатерть Улама и возвращает её клиенту. ~\newline

\item Клиент отправляет команды и отображает полученный узор.
\end{DoxyItemize}

Программа работает следующим образом:\+


\begin{DoxyEnumerate}
\item {\bfseries{Генерация спирали Улама}} ~\newline

\begin{DoxyItemize}
\item Создаётся квадратная матрица (двумерный массив) размером 100×100, заполненная пробелами. ~\newline

\item Начальная точка — центр матрицы (координаты x = 50, y = 50). ~\newline

\item Движение по спирали {\bfseries{против часовой стрелки}} с шаблоном:\+ вправо → вверх → влево → вниз. ~\newline

\item Каждой клетке присваивается натуральное число n, начиная с 1 в центре. ~\newline

\item Если число {\bfseries{простое}}, в клетку записывается {\ttfamily \texorpdfstring{$\ast$}{*}}, иначе — пробел.
\end{DoxyItemize}
\item {\bfseries{Проверка числа на простоту}} ~\newline

\begin{DoxyItemize}
\item Число 1 — не простое. ~\newline

\item 2 — простое. ~\newline

\item Чётные числа \texorpdfstring{$>$}{>} 2 — не простые. ~\newline

\item Для остальных чисел проверяются делители до √n.
\end{DoxyItemize}
\item {\bfseries{Клиент-\/серверное взаимодействие}} ~\newline

\begin{DoxyItemize}
\item Сервер слушает порт 8080. ~\newline

\item Клиент подключается к серверу и отправляет команды:\+ ~\newline

\begin{DoxyItemize}
\item {\ttfamily -\/s} — получить скатерть Улама. ~\newline

\item {\ttfamily -\/h} — справка. ~\newline

\item {\ttfamily -\/v} — версия. ~\newline

\item {\ttfamily -\/a} — автор. ~\newline

\item {\ttfamily ping} — проверка связи. ~\newline

\item {\ttfamily exit} — выход. ~\newline

\end{DoxyItemize}
\item Сервер обрабатывает команду и возвращает результат (для {\ttfamily -\/s} — строку с узором). ~\newline

\item Клиент выводит ответ на экран.
\end{DoxyItemize}
\end{DoxyEnumerate}

\hypertarget{md__r_e_a_d_m_e_autotoc_md12}{}\doxysubsubsection{\texorpdfstring{3.\+1. Спираль Улама}{3.\+1. Спираль Улама}}\label{md__r_e_a_d_m_e_autotoc_md12}
Спираль Улама — это способ укладки натуральных чисел на квадратную решётку, начиная с 1 в центре и двигаясь по спирали против часовой стрелки.

{\bfseries{Правило обхода}} (для против часовой стрелки, начиная с движения вправо):\+


\begin{DoxyCode}{0}
\DoxyCodeLine{Шаг\ 1:\ вправо\ 1\ клетку}
\DoxyCodeLine{Шаг\ 2:\ вверх\ 1\ клетку}
\DoxyCodeLine{Шаг\ 3:\ влево\ 2\ клетки}
\DoxyCodeLine{Шаг\ 4:\ вниз\ 2\ клетки}
\DoxyCodeLine{Шаг\ 5:\ вправо\ 3\ клетки}
\DoxyCodeLine{Шаг\ 6:\ вверх\ 3\ клетки}
\DoxyCodeLine{...}
\DoxyCodeLine{Длина\ шага\ увеличивается\ каждые\ 2\ направления.}

\end{DoxyCode}
\hypertarget{md__r_e_a_d_m_e_autotoc_md13}{}\doxysubsubsection{\texorpdfstring{3.\+2. Простые числа}{3.\+2. Простые числа}}\label{md__r_e_a_d_m_e_autotoc_md13}
Число {\itshape p} называется простым, если:\+


\begin{DoxyItemize}
\item p \texorpdfstring{$>$}{>} 1
\item {\itshape ∀d} ∈ \+[2, √\texorpdfstring{$\ast$}{*}p\texorpdfstring{$\ast$}{*}]\+ :\+ {\itshape p} mod {\itshape d} ≠ 0
\end{DoxyItemize}

Для оптимизации:\+
\begin{DoxyItemize}
\item Проверяем только нечётные делители (кроме 2).
\item Останавливаемся при первом найденном делителе.
\end{DoxyItemize}\hypertarget{md__r_e_a_d_m_e_autotoc_md14}{}\doxysubsubsection{\texorpdfstring{3.\+3. Координаты клетки}{3.\+3. Координаты клетки}}\label{md__r_e_a_d_m_e_autotoc_md14}
Пусть:\+
\begin{DoxyItemize}
\item size = 100
\item center = size /\+ 2 = 50
\end{DoxyItemize}

Тогда координаты клетки с числом {\itshape n}:\+


\begin{DoxyItemize}
\item {\itshape x}({\itshape n}), {\itshape y}({\itshape n}) ∈ \+[0, 99]\+
\item Начальная точка:\+ ({\itshape x\texorpdfstring{$\ast$}{*}₀, \texorpdfstring{$\ast$}{*}y\texorpdfstring{$\ast$}{*}₀) = (50, 50) для \texorpdfstring{$\ast$}{*}n} = 1.
\end{DoxyItemize}

\hypertarget{md__r_e_a_d_m_e_autotoc_md22}{}\doxysubsection{\texorpdfstring{5. Вывод}{5. Вывод}}\label{md__r_e_a_d_m_e_autotoc_md22}
В результате выполнения программы:\+
\begin{DoxyItemize}
\item Сервер запускается на порту 8080 и ожидает подключения клиентов.
\item Клиент подключается и отправляет команду {\ttfamily -\/s}.
\item Сервер генерирует {\bfseries{скатерть Улама размером 100×100}}:\+
\begin{DoxyItemize}
\item В центре — число 1 (пробел, так как 1 не простое).
\item Вокруг по спирали раскладываются числа 2, 3, 4, …
\item Простые числа отмечены символом {\ttfamily \texorpdfstring{$\ast$}{*}}.
\item Получается характерный узор с диагональными скоплениями звёздочек.
\end{DoxyItemize}
\item Клиент получает и отображает этот узор в консоли.
\end{DoxyItemize}

Таким образом, программа полностью выполняет поставленную задачу:\+ демонстрирует скатерть Улама 100×100 с выделением простых чисел звёздочками, а также предоставляет удобный интерфейс для удалённого доступа.

\item Скатерть Улама размером 100×100 успешно генерируется и передаётся клиенту.
\item Программа демонстрирует наглядный пример визуализации математического явления — упорядоченности простых чисел в спиральной укладке.
